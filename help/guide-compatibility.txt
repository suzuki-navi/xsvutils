互換性に関する情報

    以下のように最初のパラメータにバージョンを指定するオプションを渡すと、
    xsvutils の過去のバージョンによる動作になる。

        $ xsvutils -v4 ...

    将来に渡って互換性を維持する必要があるスクリプトに使用する場合は、
    このオプションを使うことで、将来の xsvutils の仕様変更にも対抗できる。
    このオプションは、先頭に指定することが必要。

    このオプションを指定した場合は、指定のバージョンをインストールされているかを xsvutils 自身がチェックし、
    インストールされている場合はそのバージョンで実行し、インストールされていない場合は
    インストールを促しエラー終了となる。
    指定のバージョンをインストールするには

        $ xsvutils -v4 --install

    のようにする。


    オプションと具体的なバージョンの対応:

        -v3: version 0.2.7
        -v4: version 0.4
        -v5: version 0.5
        ...


    注意点:

    1.  ヘルプに EXPERIMENTAL と書かれている箇所は
        まだ実装途中であったり、動作が不安定であったり、将来仕様変更する可能性の高い機能である。

    2.  ヘルプに DEPRECATED と書かれている箇所は
        廃止予定で互換性のためだけに残してある機能である。
